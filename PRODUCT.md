# ONE! Дневник — Product Context

## Product Purpose
Mobile-first PWA for ONE! International School. Students (ages 11-14) document school trips — write impressions, upload photos/videos, react to each other's posts. Teachers moderate before publishing. Parents and outsiders can browse in guest mode.

## Users
- **Primary**: Students aged 11-14. Energetic, emoji-native, short attention spans, expect Instagram-level polish.
- **Secondary**: Teachers. Need moderation tools, not flashy UI.
- **Tertiary**: Parents / guests. Read-only browsing.

## Brand
- Navy `#132245` — primary surface, headers, authority
- Orange `#ED7615` — accent, CTA, reactions, highlights
- Mint `#D6F5EE` — soft surface, badges, info blocks
- Font: Helvetica Neue CYR (Black 900 for headings, Roman 400 for body)
- Tone: energetic but trustworthy. School-official yet fun.

## Register
product

## Anti-references
- Generic school portals (boring, corporate blue)
- Childish flat cartoon UI (patronizing for 11-14)
- Glassmorphism for its own sake

## Strategic principles
- Every screen fits one hand on a phone
- Dark mode is a first-class feature
- Reactions and achievements motivate daily engagement
- Guest URLs are shareable public links
